# 项目简介

## 核心功能
- 支持图片、视频及摄像头进行花卉实例分割检测
- 提供7000张标注数据集，涵盖102类花卉
- 包含完整源码、训练好的模型及训练结果
- 支持单张图片、图片文件夹、视频文件和本地摄像头检测
- 可自定义置信度和IOU阈值
- 提供UI界面，实时显示目标位置、目标总数和置信度
- 支持检测结果保存和批量检测
- 包含项目运行教程和环境配置说明

## 适用场景
- 农业领域花卉种类识别与分类
- 植物学研究中的花卉样本分析
- 园林管理中的花卉品种识别
- 生物学教学与科研
- 智能监控系统中的花卉检测

## 技术特点
- 基于YOLOv8深度学习框架实现
- 采用Python 3.9开发，依赖opencv-python、PyQt5、ultralytics、torch1.9等库
- 提供图形用户界面，操作简便
- 集成多种输入源支持，适应不同应用场景
